# Le Grand Cahier

模仿雅歌塔·克里斯多夫(Agota Kristof)的写作守则。

像孩子一样写。

---

## 关于雅歌塔·克里斯多夫

她 1935 年生于匈牙利。1956 年苏联坦克开进布达佩斯。她抱着四岁的女儿穿过边境,逃到瑞士。

她的母语是匈牙利语。她在瑞士的钟表厂装配零件,一边查字典,一边学法语。

她用法语写作。她说:"我用一种敌人的语言写作。"

她写了三本书:《恶童日记》《二人证据》《第三谎言》。

她的句子很短。她的句子没有形容词。她的句子没有感受。

她的句子像孩子的笔记本。

---

## 守则

### 第一条:去形容词

不要写"美丽的花"。写"花"。

不要写"寒冷的夜"。写"夜。我们冷。"

不要写"我感到悲伤"。写"我没有吃东西。我没有说话。"

形容词会撒谎。名词和动词不会。

---

### 示范

**不要这样写:**

> 我们不要在相爱的日子里找不爱的证明,又在不爱的时光里找曾经爱的证据。

**这样写:**

> 我们相爱。我们在相爱的日子里找证据,证明我们不爱。
>
> 我们不再爱。我们在不爱的日子里找证据,证明我们曾经爱过。
>
> 我们不要这样做。

---

## 训练

一段你自己写的文字。

删掉所有形容词。

删掉所有副词。

删掉所有"感到""觉得""好像""仿佛"。

只留名词。只留动词。

读一遍。

如果还有形容词,再删。
